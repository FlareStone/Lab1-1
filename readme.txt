· 输入数据：
	- 先输入一个合法的多项式表达式，再输入合法的命令。
· 表达式：
	- 满足多项式基本规则；
	- 各项之间可以用“+”或“-”连接；
	- 变量名可以为长度大于1的字符串；
	- 可以用“^”表示变量的幂；
	- “*”可以省略，但两个变量间无法省略（变量的幂看作是变量的一部分）；
	- 系数和变量的乘号省略时，系数在变量前；
	- 可以包含空格。
· 命令：
	- 求值命令格式“!simplify var1=num1 var2=num2 ... varn=numn”;
	- 求值命令可以为变量赋值小数及负数；
	- 求导命令格式“!d/d var”；
	- 命令中可以包含空格。
